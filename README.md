# Valorant – moje oblíbená hra

Jednoduchý webový projekt vytvořený jako opakování práce s Gitem a GitHubem.
Stránka je věnována taktické střílečce Valorant od Riot Games.

## Autor

Jméno Příjmení

## Použité technologie

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Spuštění projektu

Otevřete soubor `index.html` v prohlížeči.

## Obsah stránky

- Informace o hře Valorant
- Oblíbení agenti
- Tipy pro začátečníky
- Interaktivní tlačítko s náhodnou hláškou
